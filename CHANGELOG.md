# Changelog

All notable changes to [buttplug-go](https://github.com/hirusha-adi/buttplug-go) are documented here.

## 1.0.0 (2026-06-13)

### Features

- Initial Go release — 1:1 port of [`buttplug-py`](https://github.com/buttplugio/buttplug-py) protocol v4 client
- `Client`, `ButtplugDevice`, `DeviceFeature`, and full message serialization layer
- WebSocket connector with request/response correlation
- Python examples ported to Go under [`examples/`](https://github.com/hirusha-adi/buttplug-go/tree/main/examples)
- Additional [`device_control_simulated_stroker`](https://github.com/hirusha-adi/buttplug-go/tree/main/examples/device_control_simulated_stroker) example
